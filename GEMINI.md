# SHIN SEONG MIN | Portfolio Project

이 프로젝트는 UX/UI 디자이너이자 주니어 개발자인 신성민의 개인 포트폴리오 웹사이트입니다. 세련된 애니메이션과 반응형 디자인을 통해 개인의 역량과 프로젝트 결과물을 효과적으로 전달하도록 설계되었습니다.

## 🚀 프로젝트 개요
- **목적**: 개인 포트폴리오 전시 및 경력 소개
- **대상**: 채용 담당자 및 협업 파트너
- **주요 컨셉**: "태도는 MAXIMUM, 디자인은 MINIMUM" - 불필요함을 덜어낸 깔끔한 UI와 강력한 인터랙션

## 🛠 기술 스택
- **Frontend**: HTML5, Tailwind CSS (CDN 사용), Vanilla JavaScript
- **Animation**: GSAP, ScrollTrigger, SplitType
- **Tools**: Python (콘텐츠 업데이트 자동화용), Figma (디자인 원본)

## 📁 디렉토리 구조
- `index.html`: 웹사이트의 메인 구조 및 콘텐츠
- `css/style.css`: Tailwind CSS 외의 커스텀 스타일 및 레이아웃 정의
- `js/script.js`: GSAP를 활용한 스크롤 애니메이션 및 상호작용 로직
- `img/`: 프로필 이미지, 프로젝트 스크린샷 등 모든 이미지 자산
- `replace.py`: `index.html` 내의 특정 섹션(예: Skills)을 자동으로 교체/업데이트하는 스크립트
- `1.md`: 디자인 수정 및 기능 확장을 위한 참고용 프롬프트 기록

## 🏃 실행 및 관리
### 실행 방법
별도의 빌드 과정이 필요 없는 정적 프로젝트입니다. 브라우저에서 `index.html` 파일을 직접 열어 확인할 수 있습니다.

### 콘텐츠 업데이트 (Skills 섹션 등)
프로젝트 내의 `replace.py` 스크립트는 `index.html`의 내용을 프로그래밍적으로 수정하는 데 사용됩니다. 대규모 레이아웃 변경이나 반복적인 기술 스택 업데이트 시 이 스크립트를 활용하거나 참고할 수 있습니다.

## 🎨 개발 컨벤션 및 디자인 원칙
1. **애니메이션**: 모든 섹션은 사용자가 스크롤할 때 `reveal` 또는 `reveal-toss` 클래스를 통해 자연스럽게 등장해야 합니다. (GSAP ScrollTrigger 기반)
2. **반응형 디자인**: Tailwind CSS 클래스를 사용하여 모바일 및 데스크탑 환경에 모두 최적화되어 있습니다.
3. **디자인 통일성**: 파란색(`text-[#2D7FF9]`) 포인트 컬러와 둥근 모서리(`rounded-2xl`) 카드를 기본 디자인 요소로 사용합니다.
4. **이미지 자산**: 새로운 프로젝트 추가 시 `img/` 디렉토리에 최적화된 이미지를 업로드한 후 `index.html`에서 경로를 참조합니다.
